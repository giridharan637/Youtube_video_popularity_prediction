About ViewProphet

ViewProphet is a modern AI-powered web application designed to help YouTube creators predict the success of their videos before they even hit the publish button. By analyzing historical data and metadata (like title length, category, and tags), our machine learning model forecasts potential views and assigns a popularity score.

Core Architecture & Tech Stack

HTML5 Tailwind CSS JavaScript ES6+ Python Flask Scikit-Learn ML Chart.js

Developed With Purpose

Built as a demonstration of robust full-stack development, modern AI dashboard aesthetics, premium glassmorphism interfaces, and frictionless machine learning API integration.
